# 崔某某某在室外公共场所无照经营案

## 案件基本信息

**被处罚单位（被处罚人）**：崔某某某

**执法部门**：**村综合行政执法局

**作出行政处罚的日期**：2026-06-20

## 行政处罚决定

当事人崔某某某于2026年6月17日在浙江某****路**号灯杆旁实施了在室外公共场所无照经营的行为，违反了《无证无照经营查处办法》第二条、第三条的规定，根据《无证无照经营查处办法》第十三条“从事无照经营的，某某由工政管理部门依照相关法律、行政法规的规定予以处罚。法律、行政法规对无照经营的处罚没有明确规定的，某某由工政管理部门责令停止违法行为，没收违法所得，并处1万某某以下的罚款”的规定，同时根据《杭州市综合行政执法行政处罚裁量权实施规定》第十二条第一款“违法行为属于《杭州市综合行政执法行政处罚裁量基准》的行为之一的，罚款数额在裁量基准确定的情节所对应的罚款幅度内确定，另有规定的除外。”和《杭州市综合行政执法行政处罚裁量基准》第十章第一条“从事无照经营的，应根据《无证无照经营查处办法》第十三条的规定予以行政处罚。未使用载货（运输）工具的，第一次到第三次，处20元以上200元以下罚款；第四次及以上，处200元以上500元以下罚款，有违法所得的，没收违法所得。”的规定，对当事人作出如下行政处罚决定：1.处以罚款人民币柒拾元整（¥70.00）。当事人自收到行政处罚决定书之日起十五日内，按照行政处罚决定书载明的履行方某某履行。
