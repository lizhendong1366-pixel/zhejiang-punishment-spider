# 浙江省行政处罚数据爬虫

## 功能说明

该爬虫可以自动抓取浙江省政务服务网的行政处罚数据，支持自动翻页和批量下载。

## 运行环境

- Python 3.8+
- Playwright
- Chromium浏览器

## 安装依赖

```bash
# 激活虚拟环境
source venv/bin/activate

# 安装依赖（已完成）
pip install playwright
playwright install chromium
```

## 使用方法

### 快速开始（抓取10页）

```bash
source venv/bin/activate
python auto_spider.py
```

### 修改抓取页数

编辑 `auto_spider.py` 文件，找到第16行：

```python
max_pages = 10  # 修改这里的数字
```

## 数据格式

CSV文件包含以下字段：

1. **序号** - 记录编号
2. **案件名称** - 行政处罚案件的完整名称
3. **处罚对象** - 被处罚的个人或单位
4. **决定书文号** - 行政处罚决定书的编号
5. **处罚决定日期** - 作出处罚决定的日期

## 技术特点

1. **自动化翻页**: 智能识别分页结构，自动点击下一页
2. **重复检测**: 自动检测重复数据，避免无限循环
3. **截图记录**: 每页自动截图，方便调试和验证
4. **错误处理**: 完善的异常处理，确保稳定运行
5. **反爬措施**: 模拟真实用户操作，降低被封风险

## 注意事项

1. **抓取速度**: 每页之间有延迟，避免请求过快
2. **网站限制**: 网站共1000页数据，请根据需要设置页数
3. **网络稳定**: 确保网络连接稳定，避免中断
4. **合规使用**: 仅供学习和研究使用，请遵守网站使用条款

## 故障排除

### 问题1: 表格加载超时

**解决**: 增加等待时间，修改 `time.sleep()` 参数

### 问题2: 无法翻页

**解决**: 检查网站结构是否改变，查看截图文件

### 问题3: 数据重复

**解决**: 脚本会自动检测并停止重复抓取
