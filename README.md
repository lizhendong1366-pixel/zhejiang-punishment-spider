# 浙江省行政处罚案例数据集

## 项目概述

本项目采集并整理了浙江省政务服务网公开的行政处罚结果信息，经过脱敏处理后形成可供学习和研究使用的案例数据集。

## 一、数据源选择理由及简介

### 1.1 选择理由
- **权威性**：浙江省人民政府办公厅主办的官方平台
- **数据丰富性**：涵盖多个执法领域
- **法律价值**：典型违法行为和处罚决定
- **公开性**：依法公开，无访问限制

### 1.2 数据源简介
- 网站名称：浙江省政务服务网行政处罚结果信息公开系统
- 网址：https://xzcf.zjzwfw.gov.cn/punishment/
- 数据内容：行政处罚决定书摘要信息

## 二、数据采集方法

### 2.1 采集架构
使用Playwright自动化网页采集框架

### 2.2 采集流程
1. 访问列表页面
2. 定位案例列表
3. 进入详情页面
4. 提取详情内容
5. 翻页处理
6. 数据保存

### 2.3 技术实现
- 反爬虫应对策略
- 动态内容加载处理
- 错误处理与重试

## 三、数据清洗方法

### 3.1 清洗目标
- 格式标准化
- 字段规范化
- 异常值处理
- 分类标签化

### 3.2 清洗流程
原始数据 → 格式检查 → 字段提取 → 内容标准化 → 关联处理 → 标签添加

## 四、脱敏处理方法

### 4.1 脱敏范围
- 个人信息（姓名、身份证、电话）
- 地理信息（详细地址）
- 企业信息（公司名称）
- 法律文书（文书号）
- 车牌号等

### 4.2 脱敏规则
- 姓名 → 替换为"某某"
- 身份证号 → 保留前后几位，中间隐藏
- 电话号码 → 保留前后几位，中间隐藏
- 地址 → 只保留区县级
- 公司名称 → 保留组织形式

## 五、数据集结构

### 5.1 目录组织
```
爬虫脚本/
```

### 5.2 文件命名
